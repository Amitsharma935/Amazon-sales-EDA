# 📊 Amazon Sales Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on an Amazon Sales dataset containing 10,000 sales records. The objective is to understand customer purchasing behavior, identify sales trends, analyze profitability, and generate business recommendations using Python.

---

## 🎯 Project Objectives

- Perform data cleaning and preprocessing
- Explore sales and profit trends
- Analyze customer purchasing behavior
- Identify top-performing categories and products
- Compare regional performance
- Study the impact of discounts on profit
- Generate business insights and recommendations

---

## 📈 Key Insights

- Electronics generated the highest sales.
- High discounts reduced overall profit.
- Certain regions outperformed others in sales.
- Top products contributed significantly to total revenue.
- Customer purchasing behavior varied across segments.

## 💡 Business Recommendations

- Increase investment in high-performing product categories.
- Optimize discount strategies to improve profitability.
- Improve sales in low-performing regions.
